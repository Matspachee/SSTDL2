Validador de Esquemas JSON

Validación de datos con esquemas JSON

Ingresa un esquema JSON y los datos a validar para comprobar si cumplen con la estructura definida.

Ejemplos de Uso

Esquema JSON:

{
  "type": "object",
  "properties": {
    "name": { "type": "string" },
    "email": { "type": "string", "format": "email" }
  },
  "required": ["name", "email"]
}

Datos JSON:

{
  "name": "Ana López",
  "email": "ana@example.com"
}

Esquema JSON:

{
  "type": "object",
  "properties": {
    "id": { "type": "integer" },
    "title": { "type": "string" },
    "price": { "type": "number" },
    "available": { "type": "boolean" }
  },
  "required": ["id", "title", "price"]
}

Datos JSON:

{
  "id": 101,
  "title": "Auriculares Inalámbricos",
  "price": 59.99,
  "available": true
}
