El programa analiza una cadena de prueba predeterminada ("10 0.1 10.2 Hola hola_ .2 0. 3hola") y mostrará los resultados según la gramática definida en el automata.

Ejemplo de Salida

10 = INT
0.1 = FLOAT
10.2 = FLOAT
Hola = ID
hola_ = ID
.2 = error
0. = error
3hola = error
